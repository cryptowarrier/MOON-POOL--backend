# MOON POOL (Backend)

## Project Description

  Plyers deposit their MOON tokens and play game in created room.
  Winner get 80% of total deposited tokens and loser will get 20%.

## Techniques

  Socket.io, Ethers.js, Typescript, Multer, Mongo aggregation
